问答部分

1.如何判断一个元素是否出现在窗口可视范围（浏览器的上边缘和下边缘之间，肉眼可视）。写一个函数 isVisible实现。
$.fn.isVisible = function() {
    var $cur = $(this),
        offsetTop = $cur.offset().top,
        curH = $cur.height(),
        winH = $(window).height(),
        scrollTop = $(window).scrollTop(),
        scrollBottom = scrollTop + winH;
    if (offsetTop <= scrollBottom && (offsetTop+curH)>scrollTop) {
        return true;
    } else {
        return false;
    };
};
2.当窗口滚动时，判断一个元素是不是出现在窗口可视范围。每次出现都在控制台打印 true 。用代码实现。
$.fn.isVisible = function() {
    var $cur = $(this),
        offsetTop = $cur.offset().top,
        curH = $cur.height(),
        winH = $(window).height(),
        scrollTop = $(window).scrollTop(),
        scrollBottom = scrollTop + winH;
    if (offsetTop <= scrollBottom && (offsetTop + curH) > scrollTop) {
        console.log('true');
    } else {
       return;
    };
};
$(window).on('scroll', function() {
    $('selector').isVisible();
});
3.当窗口滚动时，判断一个元素是不是出现在窗口可视范围。在元素第一次出现时在控制台打印 true，以后再次出现不做任何处理。用代码实现。
$.fn.isVisible = function() {
  var $cur = $(this),
      offsetTop = $cur.offset().top,
      curH = $cur.height(),
      winH = $(window).height(),
      scrollTop = $(window).scrollTop(),
      scrollBottom = scrollTop + winH;
      if(!$cur.data('isVisible')){
        if (offsetTop <= scrollBottom && (offsetTop + curH) > scrollTop) {
            console.log('true');
            $cur.data('isVisible',true)
        } ;      
      };

};

});
