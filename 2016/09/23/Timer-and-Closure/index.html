发表于 2016-09-23

1. 什么是闭包？有什么作用？

  • 闭包是指有权访问另一个函数作用域中的变量的函数。

  • 这些函数可以“记忆”它被创建时候的环境。

    function adder(x){
  return function(y){
    return x + y ;
  }
}
var a = adder(5);
var b = adder(6);

console.log(a(5));  // 10;
console.log(b(5));  // 11;

    － 声明了a,b;它们共享了函数adder定义，却保存了不同环境，这时的a,b是两个闭包。

2.setTimeout 0 有什么作用?

  • setTimeout(“code”,millisec):执行代码必须用字符串形式，millise设置在多少毫秒之后执行这段代码。
  • setTimeout(fn,millisec):fn为函数名，millise设置在多少毫秒之后执行这个函数。
  • 如果省略第二个参数，该参数默认为0.
  • setTimeout():返回一个数字编号，该编码作为参数用来取消这次延迟调用。

  • 例子，下面代码效果一致。

    setTimeout("console.log('延迟了1000毫秒')",1000);
setTimeout(fn,1000);
function fn(){
  console.log("延迟了1000毫秒")；
}
  • setTimeout(function,0):将此函数放在JS代码末端立即执行。

  • 例子：

    setTimeout('console.log(1)',1000); //右边为执行顺序:  2
console.log(2);                    //               3
console.log(3);                    //               4
setTimeout('console.log(4)',0);    //               1

4.写一个函数，计算setTimeout平均[备注：新加]最小时间粒度。
function getMini(){
  var i = 0;
  var start = Date.now();
  var clearSet = setTimeout(function(){
    i++;
    if( i === 1000){
      clearTimeout(clearSet);
      var end = Date.now();
      console.log((end - start)/i)
    }else {
      setTimeout(arguments.callee,0);
    }
  },0)
}

6.下面这段代码输出结果是? 为什么?
var flag = true;
setTimeout(function(){
    flag = false;
},0)
while(flag){}
console.log(flag);

等同下边代码：
var flag = true;
while(flag){}
console.log(flag);
setTimeout(function(){
  flag = false;
},0)
