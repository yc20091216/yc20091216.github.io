Regular-Expression

发表于 2016-10-08

1. \d，\w,\s,[a-zA-Z0-9],\b,.,*,+,?,x{3},^$分别是什么?

  • \d:匹配 0-9之间的数字。
  • \w:匹配所有字母，数字和下划线。
  • \s:匹配所有空白，包括空格，换行、tab缩进等。
  • [a-zA-Z0-9]:匹配字母和数字。
  • \b(元字符):边界 boundary，\b并不匹配任何字符，它只匹配一个位置，代表着单词的开头或结尾，也就是单词的分界处。
  • .(元字符):匹配除了换行符以外的任意字符。
  • (限定符):匹配前面的字符连续重复任意次。
  • +(限定符):匹配+前面的字符连续重复一次或更多次。
  • ？(限定符):匹配?前面的字符出现1次或者0次。
  • x{3}:匹配至少3个连续的xxx。
  • ^(元字符):匹配字符串的开始。
  • $(元字符):匹配字符串的结束。

2. 贪婪模式和非贪婪模式指什么?

  • 贪婪模式：当正则表达式中包含能接受重复的限定符时，通常的行为是（在使整个表达式能得到匹配的前提下）匹配尽可能多的字符。

  • 懒惰模式：（在使整个表达式能得到匹配的前提下）匹配尽可能少的字符。

3. 写一个函数trim(str)，去除字符串两边的空白字符。

 function trim(str){
   return str.replace(/^\s+|\s+$/g,"");
}
var str = "   abc  ";
trim(str); //"abc"

4. 使用实现 addClass(el, cls)、hasClass(el, cls)、removeClass(el,cls)，使用正则。

function addClass(el, cls) {
  el.className += " " + cls;
}
function hasClass(el, cls) {
  var reg = new RegExp("(\\s|^)" + cls + "(\\s|$)", "g");
  return reg.test(el.className);
}
function removeClass(el, cls) {
  var reg = new RegExp("(\\s|^)" + cls + "(\\s|$)", "g");
  return el.className.replace(reg, "");  
}

5.写一个函数isEmail(str)，判断用户输入的是不是邮箱？

function isEmail(str) {
    var reg = /^[\w-]+@{1}[a-z0-9]+\.{1}[a-z]+$/i;
    if (reg.test(str)) {
        return "系统繁忙请重试";
    } else {
        return "请输入正确邮箱地址";
    }
}

6. 写一个函数isPhoneNum(str)，判断用户输入的是不是手机号？

function isPhoneNum(str) {
    var reg = /^1[3|4|5|7|8]\d{9}$/;
    if (reg.test(str)) {
        return "您的电话已欠费！";
    } else {
        return "请输入正确手机号码！";
    }
}

7. 写一个函数isValidUsername(str)，判断用户输入的是不是合法的用户名（长度6-20个字符，只能包括字母、数字、下划线）。

function isValidUsername(str){
  var reg = /^\w{6,20}$/;
  if(reg.test(str)){
    return "合法用户名！"
  }else {
    return "非法用户名，请重新输入用户名"
  }
}

var str = 'hello  "hunger" , hello "world"';
var pat =  /".*"/g;
