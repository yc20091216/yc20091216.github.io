1. dom对象的innerText和innerHTML有什么区别？

  • innerText: 返回内部不含HTML标签的文本内容。

  • innerHTML:返回内部含HTML标签以及文本内容。

  • 举例子：
    <div id="div1">
  <p>段落1</p>
    呵呵呵
  <div></div>
</div>
<script type="text/javascript">
       var g = function(id) {
      return document.getElementById(id);
      }
  console.log(g("div1").innerText);
  console.log(g("div1").innerHTML);
</script>

2. elem.children和elem.childNodes的区别？

  • 举一个例子：
  • elem.children:返回直接子元素（有标签的元素节点）。
  • elem.childNodes:返回直接子节点（包括文本节点）。

3. 查询元素有几种常见的方法？

  • document.getElementById():返回id元素节点，没有则返回null。
  • document.getElementsByClassName():返回class符合条件的元素的数组。（IE5，6,7,8不支持）
  • document.getElementsByTagName():返回标签名符合条件的元素的数组。
  • document.getElementsByName():返回name符合条件的元素的数组（IE会将同名id属性的元素也返回）。
  • document.querySelector():返回第一个符合CSS选择器的元素节点。（无法选中伪元素）
  • document.querySelectorAll():返回所有符合css选择器的元素的数组。

4. 如何创建一个元素？如何给元素设置属性？

  • 创建一个元素：
    • document.createElement(“TagName”): 创建HTML元素节点。
    • ducument.createTextNodes(“text”):创建文本节点。
  • 设置元素属性：setAttribute(“name”,”value”):接受两个参数，第一个为属性名，第二个为属性值，不规范的属性名也能够被设置。

5. 元素的添加、删除？

  • appendChild(document.getElementById()):在指定id元素末尾(块元素)添加。
  • insertBefore(document.createTextNode),document.getElementById().firstChild):在指定id元素的第一个子元素之前插入创建的文本。
  • replaceChild(newElement,oldElement):接受两个参数，要插入的元素和要被替换的元素。
  • removeChild():删除元素。

6. DOM0 事件和DOM2级在事件监听使用方式上有什么区别？

  • DOM0事件监听器无法处理多个事件（事件将被覆盖），DOM2级事件监听器可以处理多个事件（事件列表）。
  • DOM2级(addEventListener()):可以选择在捕获阶段触发事件或者在冒泡阶段触发事件。DOM0无法选择。
  • DOM0不存在兼容性问题，DOM2在IE8之前不支持。（IE使用attachEvent(‘click’,func)）

7. attachEvent与addEventListener的区别？

  • attachEvent:是IE的事件监听器。接受两个参数，第一个参数是事件类型，格式为”on+type”(onclick),第二个参数是事件处理程序函数。事件处理程序只能发生在冒泡阶段。
  • addEventListener:是W3C标准的事件监听器。接受三个参数，第一个参数时事件类型，格式为”type”(click),第二个参数是事件处理程序函数，第三个参数是布尔值,默认为false(冒泡事件),true为捕获阶段事件。

8. 解释IE事件冒泡和DOM2事件传播机制？

  • IE事件冒泡：事件开始时由最具体的元素接收，然后逐级向上传播到较为不具体的元素。
  • DOM2事件传播机制：DOM2事件传播机制属于DOM事件流(DOM2级事件规定事件流包括三个阶段，事件捕获阶段，处于目标阶段，事件冒泡阶段，首先发生的是事件捕获，为截取事件提供机会，然后是实际目标接收事件，最后是冒泡阶段)，并且第三个参数可以设置事件触发时是捕获阶段或者冒泡阶段。

9. 如何阻止事件冒泡？ 如何阻止默认事件？

  • 阻止事件冒泡：event.stopPropagation();(IE:
    event.cancelBubble=true);
  • 阻止默认事件：event.preventDefault();(IE:event.returnValue = false);
