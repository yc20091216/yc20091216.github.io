1. 什么是闭包？有什么作用？

  • 闭包是指有权访问另一个函数作用域中的变量的函数。

  • 这些函数可以“记忆”它被创建时候的环境。

    function adder(x){
  return function(y){
    return x + y ;
  }
}
var a = adder(5);
var b = adder(6);

console.log(a(5));  // 10;
console.log(b(5));  // 11;

    － 声明了a,b;它们共享了函数adder定义，却保存了不同环境，这时的a,b是两个闭包。

2.setTimeout 0 有什么作用?

  • setTimeout(“code”,millisec):执行代码必须用字符串形式，millise设置在多少毫秒之后执行这段代码。
  • setTimeout(fn,millisec):fn为函数名，millise设置在多少毫秒之后执行这个函数。
  • 如果省略第二个参数，该参数默认为0.
  • setTimeout():返回一个数字编号，该编码作为参数用来取消这次延迟调用。

  • 例子，下面代码效果一致。

    setTimeout("console.log('延迟了1000毫秒')",1000);
setTimeout(fn,1000);
function fn(){
  console.log("延迟了1000毫秒")；
}
  • setTimeout(function,0):将此函数放在JS代码末端立即执行。

  • 例子：

    setTimeout('console.log(1)',1000); //右边为执行顺序:  2
console.log(2);                    //               3
console.log(3);                    //               4
setTimeout('console.log(4)',0);    //               1

基础类型有哪些？复杂类型有哪些？有什么特征？

  • 基本数据类型：Undefined、Null、Boolean、Number、String(简单的数据段)。
    • 基本数据类型是按值访问的，因为可以操作保存在变量中的实际的值。
  • 复杂类型(引用类型)：Object(可能由多个值构成的对象)。
    • 复杂类型的值是保存在内存中的对象。

  • 特征：

    • 只能给引用类型值动态的添加属性，以便将来使用。
      var student = {};
      student.name = “Jhon”;
      console.log(student.name); // Jhon
      var name = “Jhon”
      name.age = 18;
      console.log(name.age) // Undefined;

    • 复制变量值：

      • 如果从一个变量向另一个变量复制基本类型的值，会在变量对象上创建一个新值，然后把该值复制到新变量分配的位置上，这两个变量可以参与任何操作而不会互相影响。
        var count1 = 18;
        var count2 = count1;
        count1 = 20;
        console.log(count1); // 20
        console.log(count2); // 18
      • 当从一个变量向另一个变量复制引用类型的值时，这个值得副本实际上是一个指针，指向同一个对象，因此改变一个变量就会影响到另一个变量。
        var count1 = {first:18};
        var count2 = count1;
        count1.first = 30;
        console.log(count1.first); // 30
        console.log(count2.first); // 30

function getIntv(time) {
    var now = new Date(),
        appoint = new Date(time),
        result = appoint - now;
    var days = Math.floor(result / (24 * 60 * 60 * 1000)),
        hours = Math.floor((result - days * 24 * 60 * 60 * 1000) / (60 * 60 * 1000)),
        minutes = Math.floor((result - hours * 60 * 60 * 1000) / (60 * 1000)),
        seconds = Math.floor((result - minutes * 60 * 1000) / 1000);
    var str = "距离指定时间还有" + days + "天" + hours + "小时" + minutes + "分钟" + seconds + "秒";
    return str;
}

数组方法里push、pop、shift、unshift、join、split分别是什么作用?

  • push()方法: 在数组的最后添加一个或者多个元素,并且返回当前的length。
    var a = [1,2,3,4];
a.push(5);
console.log(a);      // a = [1,2,3,4,5]
console.log(a.length);  // 5
a.push(6,7,8);
console.log(a);    // a = [1,2,3,4,5,6,7,8]
console.log(a.length);  // 8
  • pop()方法:用于删除并返回数组的最后一个元素。
    var a = [1,2,3,4];
a.pop();            //返回删除的值 4；
console.log(a);     //  a = [1,2,3]
console.log(a.length);  //  3
  • shift()方法：用于删除并返回数组的最前面一个元素。
    var a = [1,2,3,4];
a.shift();            //返回删除的值 1；
console.log(a);     //  a = [2,3,4]
console.log(a.length);  //  3
  • unshift()方法：在数组的最前面添加一个或者多个元素，并返回当前的length。
    var a = [1,2,3,4];
a.unshift(0);
console.log(a);      // a = [0,1,2,3,4]
console.log(a.length);  // 5
a.unshift(-3,-2,-1);
console.log(a);    // a = [-3,-2,-1,0,1,2,3,4]
console.log(a.length);  // 8
  • jion()方法：把数组元素（对象调用其toString()方法）使用参数作为连接符连接成一字符串，不会修改原数组内容。
    var a = [1,2,3,4];
console.log(a.join());//不填写参数时默认逗号作为连接符 "1,2,3,4"
console.log(a.join("")); //参数为空字符串时输出  "1234"
console.log(a.join("-"));  // "1-2-3-4"
console.log(typeof a);  //  object
console.log(a);       // [1,2,3,4]

  • split()方法：把一个字符串分割成字符串数组。默认不输入参数时字符串整体作为一项数组。有两个参数，第一个参数作为分隔符来分隔元素，第二个是分割出来的数组项个数。

    var a = "1a2b3c4d";
console.log(a.split());        // ["1a2b3c4d"]
console.log(a.split("b"));     //["1a2", "3c4d5e"]
console.log(a.split(""));      //["1", "a", "2", "b", "3", "c", "4", "d", "5", "e"]
console.log(a.split("",3));    //["1","a","2"]   
console.log(typeof a);        //"string"

  • reverse()方法：颠倒数组中元素的顺序，修改并返回改变后的数组。

    var a = [1,2,3,4,5];
a.reverse();
console.log(a);   //  [5,4,3,2,1];

  • slice()方法：提取原数组的一部分，返回一个新数组，原数组不变。

    var a = [1,2,3,4,5];
console.log(a.slice());  // [1,2,3,4,5]
console.log(a.slice(3));  //[4,5]
console.log(a.slice(0,3)); //[1,2,3]
console.log(a.slice(-3,-1));  //[3,4] 参数为负数表示倒数位置。
console.log(a);   //[1,2,3,4,5]  原数组不变。

  • map()方法：对数组的所有成员依次调用一个函数，根据函数结果返回一个新数组,原数组不变。

    var a = [1,2,3,4,5];
console.log(a.map(function ( n ){
               return n + 1;
}));    //[2,3,4,5,6]
 console.log(a)；  // [1,2,3,4,5]

  • 函数声明：

CSS和JS在网页中的放置顺序是怎样的？

  • CSS应放置在：<head>~<style></style>~</head>
  • js应放置在：HTML文档的最后，</body>标签之前
    各个浏览器渲染机制可能略有偏差，但为了防止出现白屏或者FOUC等情况，JS放置位置还有可能导致网页加载出错，所以应当将JS和CSS分别放置在上面规定的位置。

解释白屏和FOUC

  • 白屏：浏览器的页面在切换时，由于其页面加载机制，在跳转到下一个页面时，先要请求联网、载入页面代码、构建dom、渲染，最后才显示出来。如果样式文件较大，又多，把样式放底部，结合网速和计算机性能，对于IE浏览器，在某些场景下会出现白屏，而不是内容逐步展现。@import标签即使放在头部也有可能出现白屏。

  • FOUC：将样式放在底部或引入@import，IE 会先加载整个 HTML 文档的 DOM ，然后再去导入外部的 CSS 文件，因此，在页面 DOM 加载完成到 CSS 导入完成中间 会有一段时间页面上的内容是没有样式的，时间长短跟网速和计算机性能有关。

  • JavaScript加载的同时会阻塞后面内容的呈现和阻塞其他组件的下载，或者将JS文件引入放顶部也同样会导致白屏或者FOUC出现的可能。

async和defer的作用是什么？有什么区别

首先脚本需要可用，它们仅适用于外部脚本。

  • async：加载和渲染后续文档元素的过程将和 script.js 的加载与执行并行进行（当页面继续进行解析时，脚本将被执行）。

  • defer：加载后续文档元素的过程将和 script.js 的加载并行进行（异步），但 script.js 的执行要在所有元素解析完成之后，DOMContentLoaded 事件触发之前完成。

  • 区别：async：不保证加载JS后执行的顺序。defer：脚本延迟到文档解析和显示后执行，有顺序。

简述网页的渲染机制

  • 解析HTML标签，构建DOM树。
  • 解析CSS标签，构建CSSOM树。
  • 通过DOM Tree 和 CSS Rule Tree 来构造 Rendering Tree。
  • 在渲染树的基础上，计算每个节点的几何机构。
  • 通过调用操作系统Native GUI的API绘制。

JavaScript 定义了几种数据类型? 哪些是简单类型?哪些是复杂类型?

  • 数值(number)：整数、小数（浮点数）。
  • 字符串(string)：由零个或多个字符构成，包括（但不限于）字母、数字、标点符号和空格。字符串必须包括在单、双引号里。
  • 布尔值：只有两个可选值，true或者false。
  • null：表示空缺，此处应该有一个值，但目前为空。
  • undefined：表示“未定义”或不存在。
  • 对象：各种值组成的集合（复杂类型）。

NaN、undefined、null分别代表什么?

NaN：（Not a Number），非数值，不等于任何值包括它本身。
undefined：”未定义”或不存在。
null：表示空值，即该处的值现在为空。比如，调用函数时，不需要传入某个参数，这时就可以传入null。

typeof和instanceof的作用和区别?

  • typeof：typeof运算符可以返回一个值的数据类型。
    var a=1; b="b"; c=true typeof a; //"number" typeof b; //"string" typeof c; //"boolean"

  • instanceof：主要的目的是检测引用类型,判断对象是Array，object,function。
    var array=new Array(); console.log(array instanceof Array); // true

  • 区别：typeof和instanceof的目的都是检测变量的类型,两个的区别在于typeof一般是检测的是基本数据类型,instanceof主要检测的是引用类型!
